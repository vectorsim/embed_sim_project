Three-Phase PMSM Motor Model

Interior Permanent Magnet Synchronous Motor (IPMSM) modelled in the d-q rotating reference frame. Suitable for Field-Oriented Control (FOC) co-simulation via FMU export.

Export as FMU

In OpenModelica (OMEdit or omc): 

    loadFile("ThreePhaseMotor.mo");
    buildModelFMU(ThreePhaseMotor,
                  version="2.0",
                  fmuType="cs",
                  fileNamePrefix="ThreePhaseMotor");
This produces ThreePhaseMotor.fmu ready for use with the EmbedSim FMUBlock.
